# music-front

# 项目介绍
音乐网站的前端
框架：vue2
ndoe版本:v20.10.0

使用oss对象存储（AWS S3）来上传存储音乐文件

# 功能
听歌
上传歌曲
收藏歌曲
显示歌曲字幕
显示歌曲信息
管理收藏歌曲
